# autoCtrlV
windows系统上自动疯狂输出ctrl+v。喷人之必备良器

# 使用方式
1. 下载发布的.ps1文件
2. 打开cmd或是powershell
3. ctrl+v 复制你想要复读的内容
4. 在cmd或是powershell中执行该.ps1文件（可选参数-LoopCount 表明循环次数，默认9999次）
  示例：./autoCV.ps1 -LoopCount 100
5. 执行后请在五秒内将鼠标移到想要“友好交流”的文本输入位置。比如：
